Приложение - Книга учета доходов-расходов.
В приложении можно вести учет по нескольким счетам. Каждый отдельный счет это условно отдельный пользователь.
При создании счета задается его имя и начальный баланс. Счет можно редактировать или удалить,
выделив его и нажав соответствующую кнопку. Редактировать можно все поля.
Если  создан хотя бы один счет, можно добавлять доходы и расходы. В диалоговом окне добавления
счета, расхода или дохода можно добавить несколько счетов, расходов или доходов не закрывая окна.
Для этого после заполнения всех полей необходимо нажать кнопку "Добавить ещё".
Доходы и расходы редактируются и удаляются аналогично счетам.
Можно изменять все поля у доходов и расходов при редактировании, в том числе изменять счет,
с которого списывается или на который добавляется сумма.
Данные по счетам/расходам/доходам хранятся в текстовом файле в папке "Data" в кодировке UTF-8.
Можно построить график расходов или доходов, выбрав необходимый период дат.
При выборе переключателя "категории" будет сформирована диаграмма с колонками, имеющими названия категорий.
При выборе переключателя "подкатегория" будет сформирована диаграмма для выбранной категории
с колонками, имеющими названия подкатегорий.
Высота колонки отображает расход/доход по данной категории/подкатегории за выбранный период для данного счета.
